Memory That Only Bonds On Basis

A project prompt creates a typed task graph. Raw context can swarm around it, but only reports that restrict onto the declared basis are allowed to become actionable memory.

This hosted page is a proof-of-concept visualization. The same rule can run as an external service, inside an LLM wiki workflow, or as a local library in a trusted agent stack.
